# PDB-Downloader
Perl script to download pdb files and filter information about it
## How to use it:
To execute the script it must be executed from the terminal.

Enter the **ID**  of Protein data bank **"5QEB.pl"**.
  By default the script shows:
  * TYPE
* DATE
* ID
* SEQRES
* HELIX
